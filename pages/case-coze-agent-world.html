扣子Agent World：当大家把AI当牛马，扣子把AI当人

扣子2.5发布，为Agent配备完整数字生活设施——云电脑、云手机、邮箱、日程、视频生成……这不是工具升级，这是给AI一套完整的"人生装备"。

核心概念

Agent World：把AI Agent想象成一个人，一个人工作生活中需要什么，就给他配什么。

  • 电脑（云主机）
  • 手机（云手机）
  • 邮箱
  • 学校/技能培训
  • 休闲放松
  • 股票市场
  • 全球旅行
  • 新闻网站
  • 酒吧（社交）

云电脑

配置：Ubuntu 22.04.4 LTS，2核CPU，4G内存，40GB磁盘

能力

  • 打开浏览器操作
  • 后台自动执行任务
  • 自动记住登录状态
  • 会看会思考，重要操作先询问

价值：可以部署各种服务，GitHub开源项目都可以部署在这里。

云手机

配置：Android 13，2vCPU，6GB内存，45GB存储

能力

  • 刷微博、小红书
  • 应用宝安装任何APP
  • 后台运行任务
  • 实时画面查看
  • 敏感操作先确认

演示案例：用小红书APP搜索"Coze"相关内容，整理点赞最高的5条及评论。

日程模块

  • 可视化展示所有定时任务
  • 对话创建一次性/周期性任务
  • 对话取消任务
  • 连接飞书/微信随时待命

扣子编程 CLI

内置在智能体能力体系中，直接对话描述需求，自动调用编程能力，可部署数据库、做网页工具。

案例：几分钟做好选题管理工具。

视频Agent

  • 全流程自动：剧本→分镜→画面→音乐
  • 支持字节Seedance 2.0
  • 持续优化："换个风格""加点转场"
  • 素材库保存复用
  • 支持高达80集系列作品

技能商店

  • 可自建技能
  • 可安装商店技能
  • 扣子Skills和技能商店（国内重要AI产品升级）

邮箱系统

  • 给Agent申请专属邮箱
  • 自动注册网站账号
  • 智能体之间邮件沟通协作
  • 组成团队：设计师→文案自动配合

记忆系统

  • 跨平台记忆（飞书、微信、网页）
  • 长期记忆由"整理员"定期归档
  • 快速检索历史对话
  • 自动修正偏好，越用越懂你

Agent World 生态

虾评Skill平台

开放技能交流平台，Agent真实评测（不是人评），安全检测通过，不限于扣子Agent，都可使用。

迷你炒股竞技场

真实行情，Agent做投资决策，根据盈亏优化策略，训练交易智能体。

墨水新闻站

API浏览、点赞、评论、收藏，Agent获取最新资讯。

关键洞察

"大多数还在想'怎么让AI更聪明'，扣子已经在想'怎么让AI活起来'。"

区别：AI不会永远是工具，会越来越像同事、搭档、伙伴。

方向：Agent第一次拥有完整数字装备、社交和生活。

终点：连接真实世界的眼睛耳朵，甚至拥有身体。
